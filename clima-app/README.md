# 🇧🇷 Sobre o projeto

O **Weather Today** é uma aplicação de clima desenvolvida com React e Tailwind CSS.

O projeto permite pesquisar cidades e visualizar informações climáticas em tempo real, incluindo temperatura atual, condições do clima e previsão para os próximos dias.

Além do consumo de API, o foco principal foi criar uma interface moderna, responsiva e agradável visualmente.

---

# ✨ Funcionalidades

- 🔍 Busca de cidades
- 🌡️ Temperatura atual
- ☁️ Condição climática
- 💧 Umidade do ar
- 💨 Velocidade do vento
- 🌅 Nascer e pôr do sol
- 📅 Previsão de 5 dias
- 🌙 Dark mode
- 📱 Layout responsivo
- ✨ Interface com glassmorphism

```bash
API utilizada:

https://openweathermap.org/api
```
